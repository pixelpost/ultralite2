Ultralite2
==========

The second generation core framework for Pixelpost.

Requirements
------------

* Apache 2 or highter
	* mod_rewrite

* PHP 5.3.0 or highter
	* SQLite3
	* GD2

Setup & Configuration
---------------------

1. Copy the `app` folder in your web directory.
2. Launch the install script `http://www.you.com/app/install.php`.
3. Enjoy !
