Ultralite2
==========

The second generation core framework for Pixelpost.

Requirements
------------

* Apache 2 or highter
	* mod_rewrite

* PHP 5.3.0 or highter
	* SQLite3
	* GD2

Setup & Configuration
---------------------

1. Modify and rename `./app/sample_htaccess` to `./.htaccess`.
2. Modify and rename `./private/sample_config.json` to `./private/config.json`.
